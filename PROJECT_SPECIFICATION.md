# DeskClock Project Specification

## Project Overview

DeskClock is a personal embedded systems project designed to strengthen my understanding of Computer Engineering concepts by building a real embedded device.

The project combines hardware and software to create a smart desk companion capable of displaying time, environmental data, and productivity tools.

The project will be developed incrementally, with each version introducing new functionality while maintaining a working system.

## Objectives

The objectives of this project are:

- Learn embedded systems programming using C++.
- Learn how microcontrollers interact with hardware.
- Gain experience working with sensors and displays.
- Apply software engineering principles to a personal project.
- Build a professional portfolio project for internships.

## Scope

Version 1 includes:

- Time display
- Date display
- Wi-Fi synchronization
- Temperature display
- Humidity display

Future versions may include:

- Pomodoro timer
- Alarm
- Countdown timer
- Settings menu
- Custom enclosure
